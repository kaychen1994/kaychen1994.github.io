JS 觀念 - Promise 非同步

Promise 物件代表一個即將完成、或失敗的非同步操作，以及它所產生的值。

  • Promise - MDN
  • 使用 Promise - MDN

非同步觀念

非同步行為會在所有程式碼執行完後才執行，因此看到非同步行為，執行順序就放最後。

就算 setTimeout 設定 0 秒，一樣不會改變它最後執行的順序。
